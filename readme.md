# 给钟插件

## 简介

给钟插件，玩家进服自动检测背包是否有钟，没有钟则会给玩家一个钟，建议配合钟菜单相关的插件使用。

## 安装

将`GiveClock.js`放入`plugins`文件夹后重启服务器即可。

## 功能

- 进服自动给钟并提示
- 使用游戏内指令`/getclock`或`/gc`可以获取钟
